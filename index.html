Versi v0.1.4 beta
Update Terakhir 13 Mei 2026

Riwayat Pembaruan

v0.1.4 | 13 Mei 2026 (Terbaru)
  • Smart Login Shield: Tombol chat kini tersembunyi secara otomatis saat layar login aktif atau user belum terautentikasi.
  • Interactive Chat System: Penambahan fitur Timestamp (waktu kirim) yang sinkron dengan zona waktu lokal Indonesia.
  • User Avatar Colors: Pemberian warna unik dan konsisten pada setiap nama user untuk memudahkan identifikasi pengirim.
  • BBM-Style Read Status: Indikator jumlah pembaca (Presence Count) di samping bubble chat untuk transparansi koordinasi.
  • PWA Service Worker: Registrasi sw.js untuk mendukung akses offline dan performa aplikasi yang lebih ringan.
  • System Push Notifications: Integrasi izin notifikasi browser agar pesan masuk muncul di status bar HP/Laptop meski aplikasi tertutup.
  • Permission Guard: Optimasi logika "minta izin" notifikasi agar tidak muncul berulang kali setelah diizinkan.
v0.1.3 | 13 Mei 2026
  • Async-Safe Rendering: Pemisahan proses cetak kartu jadwal dan pengecekan hak akses untuk stabilitas tampilan.
  • Post-Render Guard: Otomatisasi fungsi hak akses setiap kali tab diperbarui (mencegah akses "loss").
  • Dual-Layer Security: Sinkronisasi antara visibilitas tombol di dashboard dan filter input di dalam modal update.
  • Fix Template Error: Perbaikan penanganan parameter string pada fungsi tombol untuk mencegah layar kosong (crash).
  • UI/UX Tuning: Optimasi feedback getar dan sinkronisasi warna status yang lebih akurat.
v0.1.2 | 12 Mei 2026
  • Perbaikan header dan data scroll.
  • Perbaikan tampilan data dan silent sync.
  • Perbaikan kecil lainnya.
v0.1.1 | 11 Mei 2026
  • Pembuatan menu login dan logout.
  • Perubahan icon dan logo JDL.
  • Perubahan tombol database menjadi Akun.
v0.1.0 | 11 Mei 2026
  • Metode koding "Part" untuk efisiensi sistem.
  • Implementasi Scroll Area pada Riwayat Pembaruan.
  • Navigasi Tab Sinkron (Default: Jadwal).
v0.0.8 | 11 Mei 2026
  • Perbaikan UI header dan padding biru.
  • Implementasi ikon aplikasi fa-truck-moving.
v0.0.5 | 11 Mei 2026
