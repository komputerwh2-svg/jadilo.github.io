Versi v0.1.7 beta
Update Terakhir 20 Mei 2026

Riwayat Pembaruan

v0.1.7 | 20 Mei 2026 (Terbaru)
  • Deterministic Index Remapping: Implementasi parameter indexRender pada callback loop array filteredData.map() untuk mengunci koordinat elemen DOM secara absolut guna menghindari duplikasi dan salah sasaran pembaruan status.
  • Bidirectional Modal Bridge: Integrasi atribut data jembatan dinamis (data-index-render) pada tombol eksekusi btn-simpan-modal sebagai media komunikasi runtime antara komponen penampil jadwal dengan form data mutasi modal update.
  • Multi-Parameter Signature Refactoring: Pembaruan tanda tangan fungsi (function signature) pada bukaModalUpdate agar mampu menangkap data string esensial secara sekuensial (id, driver, wh1, wh2, statusAktif, ekspedisi, kota, indexRender).
  • Injection String Sanitize: Pengamanan karakter khusus (escape character handling) pada pengiriman argument string nama ekspedisi dan nama kota di dalam penulisan sintaks templat literal HTML (onclick handler) untuk mengeliminasi potensi broken string akibat tanda petik satu.
  • Floating Chat Button Repositioning: Kalibrasi ulang tata letak z-index dan utilitas spacing (bottom/right padding) pada tombol *Floating Action Button* (FAB) Chat untuk mencegah tumpang tindih dengan konten kartu rincian armada serta menjaga aksesibilitas navigasi bawah.
  • Ambiguous Gesture Pointer Filtering: Penyempurnaan modul inisialisasiDragChat lewat kalkulasi delta koordinat absolut (threshold 5px) guna membedakan secara instan antara ketukan tidak sengaja (micro-tremor jari) dengan aksi seret, memulihkan fungsionalitas ketuk-langsung pada perangkat smartphone tanpa hambatan lock-drag.
v0.1.6 | 20 Mei 2026
  • Full-Stack Real-Time Dark Mode Support: Implementasi varian utilitas kelas dark: Tailwind CSS secara menyeluruh pada komponen manipulasi DOM dinamis berbasis JavaScript.
  • Adaptive State Injection: Restrukturisasi fungsi load jadwal dan tampil pesan kosong untuk menjamin warna indikator sinkronisasi cloud server, teks pemuatan, dan pesan galat offline tetap terbaca dengan kontras tinggi saat mode malam aktif.
  • Dynamic Container Theme Remapping: Integrasi pemetaan warna reaktif di dalam daftar jadwal untuk membedakan secara visual kartu jadwal berstatus Aktif (aksen biru/gelap malam) dan Arsip (aksen slate redup).
  • Polymorphic Status Card: Penyesuaian skema variabel warna dinamis di dalam pilih jadwal, mencakup transformasi adaptif kartu detail rincian driver untuk status DATANG, PENDING, dan BELUM tanpa merusak pendaran warna latar belakang (opacity reduction).
  • Interactive Component Optimization: Penyelarasan visual tombol aksi "Buka" dan "Ubah Status", lencana muatan gudang (WH-1 & WH-2), serta badge animasi melompat [REVISI] agar menyatu secara estetis dengan latar belakang gelap gulita (anti-silau).
v0.1.5 | 19 Mei 2026
  • Top-Right Action Layout: Mereposisi tombol "Ubah Status" secara presisi ke pojok kanan atas, sejajar dengan baris nama kota dan tepat berada di bawah informasi Total Muat Karton.
  • Three-State Queue Indicators: Integrasi pesan indikator otomatis bersuarat pada status muat gudang (WH-1 & WH-2) saat jam selesai masih kosong:
    • Status DATANG → "Menunggu Antrian..."
    • Status PENDING → "Muat Besok"
    • Status BELUM → "Belum Datang"
  • Unified Box Vertical Layout: Penyusunan rincian muatan WH-1 dan WH-2 secara vertikal dengan penggabungan teks label dan nilai muatan ke dalam satu kotak lencana (*badge system*).
  • Inline Timestamp Badge: Penempatan data waktu/jam selesai muat operasional lapangan secara langsung (*inline*) di samping kanan lencana masing-masing gudang.
  • Layout Bottom Clean Up: Pembersihan area log status kedatangan armada di bagian bawah kartu rincian agar visual terlihat lebih lega, rapi, dan fokus.
v0.1.4 | 13 Mei 2026
  • Smart Login Shield: Tombol chat kini tersembunyi secara otomatis saat layar login aktif atau user belum terautentikasi.
  • Interactive Chat System: Penambahan fitur Timestamp (waktu kirim) yang sinkron dengan zona waktu lokal Indonesia.
  • User Avatar Colors: Pemberian warna unik dan konsisten pada setiap nama user untuk memudahkan identifikasi pengirim.
  • BBM-Style Read Status: Indikator jumlah pembaca (Presence Count) di samping bubble chat untuk transparansi koordinasi.
  • PWA Service Worker: Registrasi sw.js untuk mendukung akses offline dan performa aplikasi yang lebih ringan.
  • System Push Notifications: Integrasi izin notifikasi browser agar pesan masuk muncul di status bar HP/Laptop meski aplikasi tertutup.
  • Permission Guard: Optimasi logika "minta izin" notifikasi agar tidak muncul berulang kali setelah diizinkan.
v0.1.3 | 13 Mei 2026
  • Async-Safe Rendering: Pemisahan proses cetak kartu jadwal dan pengecekan hak akses untuk stabilitas tampilan.
  • Post-Render Guard: Otomatisasi fungsi hak akses setiap kali tab diperbarui (mencegah akses "loss").
  • Dual-Layer Security: Sinkronisasi antara visibilitas tombol di dashboard dan filter input di dalam modal update.
  • Fix Template Error: Perbaikan penanganan parameter string pada fungsi tombol untuk mencegah layar kosong (crash).
  • UI/UX Tuning: Optimasi feedback getar dan sinkronisasi warna status yang lebih akurat.
v0.1.2 | 12 Mei 2026
  • Perbaikan header dan data scroll.
  • Perbaikan tampilan data dan silent sync.
  • Perbaikan kecil lainnya.
v0.1.1 | 11 Mei 2026
  • Pembuatan menu login dan logout.
  • Perubahan icon dan logo JDL.
  • Perubahan tombol database menjadi Akun.
v0.1.0 | 11 Mei 2026
  • Metode koding "Part" untuk efisiensi sistem.
  • Implementasi Scroll Area pada Riwayat Pembaruan.
  • Navigasi Tab Sinkron (Default: Jadwal).
v0.0.8 | 11 Mei 2026
  • Perbaikan UI header dan padding biru.
  • Implementasi ikon aplikasi fa-truck-moving.
